# NoClip Mod — GTA SA Android

## كيفية التركيب

### 1. ما تحتاجه
- **GTA SA Android** (نسخة قديمة تدعم CLEO، مو آخر إصدار من Play Store)
- **CLEO SA** مثبت على هاتفك (من cleo.li)
- **Sanny Builder 3** على الكمبيوتر لعمل الـ Compile

### 2. Compile الملف
1. افتح **Sanny Builder 3**
2. اختر: `Options → Language → GTA SA` وتأكد من وضع **SA Mobile**
3. افتح ملف `noclip.s`
4. اضغط **F6** للـ Compile
5. ستحصل على `noclip.csi`

### 3. نقل الملف للهاتف
ضع `noclip.csi` في:
```
/sdcard/Android/data/com.rockstargames.gtasa/files/CLEO/
```

### 4. تفعيل الـ Mod في اللعبة
- **هبّط الشاشة من الأعلى للأسفل** لفتح قائمة CLEO
- ابحث عن **"noclip"** في القائمة
- اضغط عليه لتفعيله ✅
